# Assets

Chess pieces, UI icons, and sound effects.

## Structure

- `pieces/` - Chess piece SVG/PNG sets (matte vector style)
- `icons/` - UI icons for buttons and controls
- `sounds/` - Move sounds, check alerts, etc.

## Asset Guidelines

All assets must meet:

- Permissive licensing (MIT, CC0, or equivalent)
- Clear attribution where required
- High readability at various sizes
- Consistent with neomorphism + glassmorphism design

See [architecture.md](../source-docs/architecture.md) for asset sourcing
strategy.
